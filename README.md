# 🧪 1週間英語ラボ

## 🌐 本番URL（Web/モバイル共通・無料）

**https://toumorokoshi118513-boop.github.io/english-lab/**

- **スマホをモバイルアプリ化**: 上のURLをスマホで開く → 共有ボタン →「ホーム画面に追加」(iPhone) / メニュー→「アプリをインストール」(Android)。フルスクリーンのアプリとして起動し、**オフラインでも動きます**（PWA）
- **データは端末ごと**に保存されます（ブラウザのlocalStorage）。PCとスマホでデータを共有したい場合は、⚙️設定の「バックアップ(JSON)」→もう片方で「復元」、またはスプシ同期を使ってください
- 更新方法: このリポジトリの main に push すると自動反映（GitHub Pages）

英語インフルエンサーの学習法を1週間ずつ実践し、テスト結果・継続率・実感などを共通指標で
比較・可視化する記録ツール（運営者専用MVP / フェーズ1）。**サーバー・DB不要・全部無料**。

## 特徴

- **単一HTMLファイル** — `index.html` をブラウザで開くだけ。データは端末内(localStorage)に自動保存
- **オフライン動作** — グラフ（棒／折れ線／レーダー）も外部ライブラリなしで描画
- **スプシ管理** — CSV書き出し or Google Apps Scriptで自動同期（どちらも無料）
- **アプリ内テスト（自動出題→自動採点→自動保存）** — 内蔵問題バンク（語彙/文法/熟語/読解・API不要）またはAI生成で出題。受けるだけで実践前/後テストに自動保存。毎回ランダム出題で暗記による水増しを防止
- **良い/悪い自動判定** — 前後テスト＋記録から `◎とても良い/○良い/△ふつう/✕いまひとつ` を根拠つきで自動判定（API不要）
- **無料LLM分析** — Groq / Gemini のAPIキーで結果を自動分析（なぜ伸びた・過去比較・次の一手）。⚙️設定に「🔌接続テスト」ボタンあり
- **SNS画像出力** — Instagram正方形/縦長/ストーリー・X・YouTube各サイズをPNGで生成
- スマホ最適化・総合スコア自動算出・比較/ランキング・週次レビュー

## 使い方（推奨: ワンクリック起動）

**`英語ラボを開く.command` をダブルクリック** してください。ローカルサーバー経由でブラウザが開きます。
これが一番確実です（ターミナルの窓が1つ開いたままになります。閉じると停止するので開いたまま使ってください）。

> 初回だけ「開発元を確認できません」と出たら: Finderで `英語ラボを開く.command` を右クリック →「開く」→「開く」。

### ⚠️ index.html を直接ダブルクリックしないでください

`index.html` を直接開く（`file://`）と、**ブラウザによってはデータ保存(localStorage)がブロックされ、
入力が保存されず「色々動かない」状態**になります（特に Safari）。必ず上の `.command` から起動してください。
その状態のときはページ上部に赤い警告バーが出ます。

手動でサーバーを立てる場合: `cd english-lab && python3 -m http.server 8731` → http://localhost:8731/index.html

## 無料LLM分析のセットアップ（任意）

AI分析・AIテスト生成を使う場合のみ必要です（内蔵テストと良い/悪い判定はAPIなしで動きます）。

⚙️設定 →「🤖 AI分析（繋がるAPIを自動使用）」に、**持っているキーだけ**入れます（1つでOK）。
分析時は**入れたキーのうち実際に繋がったものを自動で使います**。手動でのプロバイダ切替は不要です。

- **Groq**: https://console.groq.com/keys
- **OpenRouter**: https://openrouter.ai/keys
- **Gemini**: https://aistudio.google.com/apikey

複数入れておくと、片方が混雑/上限（429）でも自動でもう片方に切り替わって解析が通ります。各プロバイダ内でも複数モデルを自動で試します。「🔌接続テスト（全キー）」で、どのキーが繋がるか一覧確認できます。分析結果の下に実際に使ったモデル名（例: `gemini-2.0-flash`）が出ます。
キーは端末のブラウザ内のみに保存され、分析時にのみ各社APIへ直接送信されます（当ツールの外部サーバーは経由しません）。

### 429（レート/クォータ上限）が出る場合 — よくあります

429は「あなたの使用回数」ではなく、**無料枠の共有上限やアカウントのクォータ状態**で出ます（1回も使っていなくても起こります）。

- **Gemini** は新規アカウントで無料枠がまだ有効化されていないと初回から429になりがち（回復は数十分〜翌日）
- **Groq** は分あたりの上限が低く、混雑時は初回でも429

**一番の対処は「プロバイダを切り替える」**こと。Groq / OpenRouter / Gemini はそれぞれ別サービス＝別枠なので、片方が429でも別方が通ります。特に **OpenRouter が比較的安定**です。

### その他の「動かない」原因

1. プロバイダとキーの発行元が不一致（例: Groqを選んでGeminiのキーを貼っている）→ 一致させる
2. キー前後に空白・改行が混入 → 貼り直す
3. Groqがブラウザからブロック（通信失敗/CORS）→ OpenRouter か Gemini に切替

> AI分析はあくまで“おまけ”です。**内蔵テストの自動採点・良い/悪い判定・比較・ランキング・グラフ・SNS画像は、APIキー無しで全て動きます。**
